About the Radar

How to Use the Radar

Quadrants

The radar is divided into four quadrants representing different categories of technology:

  • Techniques — These include elements of a software development process, such as experience design, and ways of structuring software.
  • Platforms — Things that we build software on top of: cloud providers, operating systems, or generic kinds of platforms like serverless.
  • Tools — Components such as databases, software development tools, and other supporting tools.
  • Languages & Frameworks — Programming languages like TypeScript and Python, and frameworks like React and FastAPI.

Rings

Each quadrant is divided into four rings, indicating what stage in adoption we believe the technology is at:

  • Adopt — Technologies we have high confidence in to serve our purpose. Technologies with a usage culture in production environments, low risk, and recommended to be widely used.
  • Trial — Technologies that we have seen work with success in projects to solve real problems. First serious usage experience that confirms benefits and can uncover limitations. Trial technologies are slightly more risky.
  • Assess — Technologies that are promising and have clear potential value. Worth exploring with the goal of understanding how it will affect your enterprise. Requires monitoring.
  • Hold — Technologies not recommended to be used for new projects. Technologies that we think are not worth further investment. Don't start new projects with these and consider replacing in existing projects.

Status Indicators

Items on the radar may have status indicators showing movement:

  • New — First appearance on the radar this edition.
  • Moved In — Technology has moved closer to the centre (e.g., from Trial to Adopt).
  • Moved Out — Technology has moved away from the centre (e.g., from Trial to Hold).
  • No Change — Position unchanged from previous edition.

Themes

For each edition of the Technology Radar, we look for patterns emerging from the technologies we discuss. These patterns form the themes that represent the most significant trends we're observing.
